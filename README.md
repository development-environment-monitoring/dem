# DEM - Development Environment Monitoring

Este repositório implementa o DEM, uma plataforma para monitorar a conformidade técnica de ambientes de desenvolvimento.

## Documentação por assunto

- O que é o DEM e para quem serve: [o-que-e-o-dem.md](o-que-e-o-dem.md)
- Arquitetura técnica detalhada: [arquitetura.md](arquitetura.md)

## Requisitos

- Node.js 18+
- npm 9+
- Python 3.10+ (para o cliente)

## Primeiro acesso

- Abra a tela de login no frontend.
- Cadastre o primeiro usuário.
- O primeiro cadastro vira ADMIN automaticamente.
- Usuários seguintes entram como NORMAL.
